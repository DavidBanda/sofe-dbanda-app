# sofe-dbanda-app

## Prerequisites

You will have to have previously installed

[Node.js](https://nodejs.org/en/)

[EMBER CLI](https://ember-cli.com)

## Installation

```bash
git clone https://github.com/DavidBanda/sofe-dbanda-app
```

```bash
cd sofe-dbanda-app
```

```bash
npm install
```

## Run Project

```bash
ember server
```

You can access it at (http://localhost:4200)
